
# Steel Similarity Engine

Сервис для сравнения марок сталей по химическому составу и механическим свойствам.

Результат — коэффициент похожести от 0 до 1.


##  Как работает

-   сравнение химических элементов через диапазоны (min/max)
-   сравнение механических свойств (например, предел текучести)
-   взвешивание параметров (C важнее Mn и т.д.)
-   игнорирование отсутствующих значений (null)

##  Формула

    similarity = Σ(similarity × weight) / Σ(weight)

##  Функции

-   getSteel(name) — получить сталь
-   compareRange(a, b) — сравнить диапазоны
-   compareChem(a, b) — сравнить химию
-   calculateSimilarity(a, b) — полное сравнение
-   findSimilar(name) — найти похожие стали


## Особенности
-   работает с неполными данными
-   поддерживает веса элементов
-   результат нормализован (0–1)
